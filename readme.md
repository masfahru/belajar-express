# Belajar Express.js

## Prasyarat

- Telah terinstall Node.js versi 20 atau paling baru
- VSCode sebagai code editor

## Cara menjalankan

- download zip project kemudian ekstrak zip ke suatu folder, atau lebih cepat gunakan git clone: `git clone https://github.com/masfahru/belajar-express.git`
- Buka VSCode, buka folder project
- Buka terminal dari dalam VSCode, jalankan `npm install`
- Install VSCode Extension: [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) dan [Postman](https://marketplace.visualstudio.com/items?itemName=Postman.postman-for-vscode), (optional) [Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)
- Pada terminal, jalankan `npm run dev` untuk memulai development
